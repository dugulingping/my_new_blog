#  应收账款账务核算
#企业财务会计笔记/资产/应收账款#
--- 
  应收账款是商业企业在对外销售商品、材料以及提供劳务中形成的应收且未收回的货款、增值税、代购货方垫付的包装费、运杂费等款项。

### 取得  
```
借：应收账款 
	贷：主营业务收入
		应交税金 —— 增值税（销项税）
		现金
```

## 现金折扣
### 含义
`这里有张图，后补！`
* 对现金折扣的会计处理通常有两种方法，即总价法和净价法。我国目前则采用总价法。
1. `总价法`（实际发生的现金折扣，作为财务费用计入当期损益）
* 发出商品
```
借：应收账款（总价 + 销项税额
    贷：主营业务收入（总价）
    贷：应交税金 —— 应交增值税（销项税额）
```
* 在折扣期收到款项
```
借：银行存款（净价 + 销项税额   
        财务费用（总价 — 净价）
    贷：应收账款（总价 + 销项税额）—— 某企业  
```
* 过了折扣期收到款项：
```
借：存款（总价 + 销项税额）
    贷：应收账款（总价 + 销项税额）—— 某企业
```

## 坏账损失，坏账准备
#企业财务会计笔记/资产/减值业务#
### 概念
见书本
## 确认
* 债务人破产或死亡，用其破产财产或遗产偿还后，确实不能收回的部分；
* 因债务单位撤消、资不抵债或现金流量严重不足，确定不能收回的；
* 严重的自然灾害导致债务单位停产而在短时间内无力偿付债务，确实无法收回的；
* 债务人逾期三年尚未归还的应收账款，有证据表明确实无法收回的。

## 坏账准备计提方法
企业坏账损失的处理有直接`转销法`和`备抵法`两种。现行企业会计制度规定，企业的坏账损失`必须采用备抵法。` 所谓备抵法就是企业按一定的原则和方法对可能发生的坏账损失提取相应的准备资金，在实际发生坏账损失时冲减坏账准备金并同时转销应收账款的数额。备抵法在计算坏账准备的提取金额时又有几种计算方法，下面介绍常用的三种：
* 账龄分析法：按照应收账款账龄的长短来估计坏账损失并确定应提取的准备应收账款余额百分比法：按年末应收账款余额的一定比例来估计坏账损失并确定应提取的准备金额。
- - - -
* 销货百分比法：按企业赊销金额的百分比来估计坏账损失并确定应提取的准备金额。
- - - -
* 个别认定法：对每一项应收款项进行分析，估计可能发生的坏账损失，计提坏账准备的一种方法
---i
* `应收账款余额百分比法`：按年末应收账款余额的一定比例来估计坏账损失并确定应提取的准备金额。
- - - -
## **应收账款余额百分比法计算**
* `坏账准备年末余额 = 应收账款的年末余额 × 规定的计提比例 `
* `本年应计提和冲销的坏账准备金 = 本年年末坏账准备的余额 - 上年年末坏账准备余额 + 本年发生的坏账损失 -已经确认为坏账又在本年收回的数额`
### 应收账款余额百分比法
* 当计算的结果为正数，为本年应计提的坏账准备金额 
```
借：资产减值损失  ——计提的坏账准备
	贷：坏账准备
```
* 当计算的结果为负数，为本年应冲销的坏账准备金额  
```
借：坏账准备
    贷：资产减值损失  ——计提的坏账准备
```
* 坏账损失确认
```
借：坏账准备
    贷：应收账款
```
* 以前年度确认的坏账，在本年收回
```
分录一：借：应收账款
			贷：坏账准备     
分录二：借：银行存款
			贷：应收账款
```
