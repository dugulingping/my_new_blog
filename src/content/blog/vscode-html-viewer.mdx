最近找工作(前端岗)，刷面试题，觉得还是vscode写代码舒服，无论是写博客还是调试node.js甚至是自带的git都用的超级爽~

于是就想搞个全能的html运行环境(包括实时预览和在浏览器中打开)

## 实时预览

都说到这了，你不会还以为是vs自带的功能把。对，没错，就是通过一个叫做`HTML Preview`的插件实现的。在应用商店安装就行啦~

安装好之后，按`Ctrl+Shift+V`可以打开语预览，按`Ctrl+k之后再单独按下v`就可以自动在右侧分栏打开预览。

## 浏览器打开

也是插件实现的，名字叫`open in browser`安装就行了，现在的vscode也不用像以前一样安装插件需要重载，安装完毕之后按`Alt+B`在默认浏览器中预览，按`Shift+Alt+B`可以自己选则浏览器预览。

## 关于html css js的代码补全
插件名：`HTML CSS Support`。
貌似支持好多前端语言的，具体看插件介绍。
