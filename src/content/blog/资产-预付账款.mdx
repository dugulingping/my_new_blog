`企业财务会计笔记/资产/应收账款`
---
## 性质
借方登记预付的货款
	贷方登记转消的货款
借方余额表示尚有预付货款
	贷方余额表示预付的货款不足等待补足
---
## 核算
1. 预付款时
```
借：预付账款
	贷：银行存款
```
2. 收到材料时
```
借：原材料
	 应交税费  ——应交增值税(进)
贷：预付账款
```
3. 补足货款时
```
借：预付账款
	贷：银行存款
```
4. 退回余款时
```
借：银行存款
	贷：预付账款 
```
5. 公司预付货款后，无法收到货物也无法收回所预付款项，将已预付款项转出：
```
借：其他应收款  ——某某公司
贷：预付账款  ——某某公司
```
