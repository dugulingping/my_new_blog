这个主题已经内置并配置好了 [@astrojs/mdx](https://docs.astro.build/zh-cn/guides/integrations-guide/mdx/) 继承。如果你不想使用 MDX，也可以在 `astro.config.mjs` 配置文件中将其移除。

## 为什么选择 MDX?

MDX 是一种特殊的 Markdown，它支持嵌入 JavaScript 和 JSX 语法。这让你能够 **在 Markdown 内容中混入 JavaScript 代码和 UI 组件**。

这正是实现交互式图表、警告框或者你想要的“服务状态挂件”的关键。

## 交互示例

下面演示如何在 MDX 中导入并使用一个 UI 组件。  
当你点击下面的按钮时，它会触发一个原生的 JS 弹窗。

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('你点击了组件！')">
    点击这个嵌入在 MDX 中的组件
</HeaderLink>
