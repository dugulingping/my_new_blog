## linux中dns的问题

今天用电脑给树莓派开了个热点,然后发现apt install用不了.无法连接服务器????? 然后ping baidu.com,也不行,遂ping 114.114.114.114,发现可以ping通.确定是dns服务器的问题

#### 解决办法

```shell
$sudo nano /etc/resolv.conf
#文件内容
#第一个nameserver是主dns，第二个nameserver是次dns
# Generated by resolvconf
domain mshome.net
nameserver 192.168.137.1
```

修改为

```shell
# Generated by resolvconf
domain mshome.net
nameserver 192.168.137.1
nameserver 114.114.114.114
```

保存退出 ping baidu.com 能ping通了
